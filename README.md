# 네이버 지도 api 를 활용한 웹 사이트

### 개발 계획
* 네이버 지도 즐겨찾기 기능, 네이버 지도 api 활용 하여 장소에 대해 comment 하는 웹사이트 

### 개발 환경 
* 사용 언어 : JDK 11.0.15
* 프레임 워크 : Spring, MyBatis, JDBC
* 데이터 베이스 : MySQL
* 개발 환경 : eclipse, VSCode, Apatch Tomcat, Github

### 모델링
* MVC2 패턴 

### 기능
* 등록한 장소 마커로 위치 출력
* 네이버 지역 검색
* 네이버 지역 검색을 통해 검색한 장소에 comment 작성
